梦华小院 · 产品包装思路文档

产品名：梦华小院 Slogan：种什么得什么 定位：李子柒式田园茶饮生活模拟器（种植向） 日期：2026-04-28

一、产品定位

1.1 核心设定

产品名：梦华小院
Slogan：种什么得什么
定位：李子柒式田园茶饮生活模拟器（⭐ 种植向）
文化源头：北宋·孟元老《东京梦华录》（公共领域古籍）

玩家身份：来到汴梁、拥有一座小院的掌柜（穿越者）
核心行为：在后院种植食材 → 在前厅调制茶饮甜品 → 接待客人过日子
情感主线：从市集小摊起步 → 雅士常来 → 贡品订单 → 触碰上古传说
视觉风格：温暖质朴的田园风（非精致高端，非Q版卡通——有烟火气的好看）
目标用户：下沉市场为主 + 向往慢生活的打工人 + 喜欢李子柒式内容的用户

1.2 文化内核

梦华小院的文化内涵 = 展示而非考据，自信而不晦涩：

  • 对中国茶文化的了解与展示（宋代点茶/熟水/果子体系）
  • 对宋元时期花卉、水果、本草等材料的了解（每种食材都有典籍出处）
  • 食材名字取自《诗经》《本草纲目》《梦华录》等古籍，体现命名美学
  • 通过"穿越者+丝绸之路贸易"合理解释食材多样性（西域/南洋/西洋来物）
  • 功能文案全白话，只有食材名和成品名引经据典——高级感来自名字，不来自晦涩

一句话定位：梦华小院——种什么得什么。 不是种花游戏，不是纯经营游戏，是一座小院的慢生活模拟器，种的是自己种的，喝的是自己泡的。

二、核心玩法结构

2.1 玩家收集什么

收集对象 是什么 驱动力
① 食材/植物 鲜果 + 花卉 + 本草（~514种） "我种齐了！" — 种植向核心
② 茶饮/甜品成品 底卡+三槽位食材组合（数千组合） "我是茶艺大师！" — 深度追求

2.2 玩家做什么

行为 说明 复杂度
种植 后院播种→浇水→采摘，产出三大类食材（鲜果/花卉/本草） ★☆☆
卖熟水（快轨） 居民来买熟水 → 选食材→一键制作并交付 ★★☆
做茶饮/甜品（精轨） 选底卡（茶具+底底）+ 三槽位食材搭配 → 深度调制 ★★★

2.3 两层图鉴

图鉴 内容 特点
🌱 食材图鉴 ~514种食材（鲜果/花卉/本草） 唯一种植收集核心；每种有古雅名+典故+诗词+产地故事
🏆 茶饮图鉴 数千种成品组合 深度制作追求；由底卡+槽位组合产生

关于熟水：每种食材对应唯一一种熟水（如荔枝→荔子饮），有多少食材就有多少熟水。熟水是居民订单的交付载体，不是独立图鉴——玩家在图鉴中收集的是食材本身。

2.4 双轨制经营

轨道 名称 行为本质 目标用户
快轨 熟水订单（居民订单） 单一食材 → 一键制作并交付熟水订单 全部玩家
精轨 雅客/宫宴订单 底卡 + 三槽位食材深度组合制作 进阶玩家

卖熟水的完整流程： - 居民NPC提出需求（"我要3杯紫苏饮""来2碗薄荷汤"） - 玩家从已有食材中选择对应的 → 一键制作并交付，获得铜钱 - 不表现制作过程——UI上通过图标/名称的强关联性让玩家直观感知"紫苏叶→紫苏饮" - 快轨的核心是效率：选完即完成，日常循环顺畅

与精轨的关键差异：快轨一键完成无动画；精轨有15-25秒李子柒式调制动画。快=效率，精=体验。

2.5 双成长线

成长线 名称 来源 追求方向
① 🌱 种植等级 / 园丁称号 播种/浇水/采摘/施肥 解锁稀有种苗 / 提升产量 / 扩大种植上限 / 解锁全食材图鉴
② 🍵 茶艺等级 / 茶师称号 熟水+茶饮甜品制作 解锁更多底卡 / 提升成功率 / 茶坊升级 / 解锁全茶饮图鉴

两条线独立推进，覆盖不同偏好： - 只爱种不爱做 → 园丁路线专精 - 只爱做不爱种 → 茶师路线专精 - 两样都爱 → 双修完整体验

四、设计原则

4.1 七大原则

# 原则 含义 示例
P1 术语宋化 UI文案使用宋代语汇 水滴→清泉、种子→种苗、背包→茶箱
P2 叙事包裹 功能描述升级为场景化叙述 "看广告缩短时间"→"勤心呵护，待时辰到则瓜熟蒂落"
P3 玩法不变 底层数值逻辑保持一致，只改表皮 订单结构/产出公式/VIP权益不变
P4 体验升维 不改机制的前提下提升沉浸感 设置页→账房先生整理账册背景
P5 ⭐ 种植为本 种植不是材料的来源，而是核心乐趣本身 食材有独立图鉴页/植株可展示/种植有专属经验线
P6 文案白话化 操作/说明/提示全用大白话；文雅词汇仅限食材名和成品名 按钮写"种下去"不写"播种"；但食材叫"离支""丹若""水芝"——名字本身就是高级感来源
P7 ⭐ 文化自信（不是文青） 文化内涵 = 知识展示而非门槛设置 玩家不需要懂茶文化才能玩，但玩了之后自然了解。边玩边涨知识，不费力的文化获得感

4.2 禁止事项

  • ❌ 不引入与现代不符的科技概念
  • ❌ 不破坏原有的数值平衡体系
  • ❌ 不增加新的学习成本（新术语必须直觉可理解）
  • ❌ 不使用过于生僻的文言（目标玩家不是宋史专家）
  • ❌ 不在操作按钮/提示中使用文雅词汇——白话和白话，雅名留给食材和成品名
  • ❌ 不将种植物降级为纯粹的材料（每种食材都应有存在感、故事感和收集价值）

六、重点系统说明

6.1 食材系统（种植向核心）

三大类食材

类别 数量 角色 命名特点
鲜果类 ~188种 主料（主鲜果槽） 古籍雅名：玉李、离支（荔枝）、丹若（石榴）、水芝（莲子）
花卉类 ~163种 点缀/部分辅料 古典花名：辛夷、荼蘼、木槿、菖蒲
本草类 ~163种 辅料（辅本草槽） 中医药名：蕃荷（薄荷）、国老（甘草）、广陈皮

命名体系：每种食材采用古雅名分级体系——市井级用亲切俗名（蜜桃/青杏），雅士级取古籍原名（玉梨/离支），贡品/上古级加入仙境意象（瑶桃/碧霞橙）。所有名字均可在《诗经》《本草纲目》《梦华录》等典籍中找到出处。

四级稀有度

等级 数量 代表 感觉
市井级 ~172种 蜜桃/青杏/金橘 亲切实在，小镇集市常见
雅士级 ~145种 玉梨/离支/卢橘 有品位，读书人知道的好名字
贡品级 ~110种 丹若/骊珠/水芝 宫廷御用级别
上古传说级 ~87种 瑶桃/昆仑瓜/蓬莱橘 近乎神话，穿越秘方才用的东西

食材多样性解释（穿越者叙事）

为什么游戏里有514种食材，远超宋代实际？ - 玩家身份是穿越者，天然带来后世知识 - 北宋汴梁是国际都市，丝绸之路贸易畅通无阻 - 西域来的（葡萄/石榴/胡桃）、南洋来的（芒果/椰子/榴莲）、西洋来的（番茄/玉米/番薯）都可以合理出现 - 不需要完全符合史实，文化自信不等于历史考据

6.2 熟水系统（快轨·居民订单）

  • 定义：宋代"饮子/熟水"品类——单一种食材熬制的清凉饮品
  • 对应：《梦华录》中赵盼儿卖的紫苏饮子、薄荷饮、桂花酸梅汤等
  • 规则：1种食材 = 1种熟水（严格1:1对应）
  • 操作方式：一键制作+贩卖，不表现制作过程。UI上通过图标/名称强关联表达"紫苏叶→紫苏饮"的对应关系
  • 角色：日常循环的核心行为（高效），引导玩家从"种"过渡到"做"
  • 不是独立图鉴：它是订单交付载体，收集对象是底层食材
  • 与精轨的区别：快轨=一键无动画（效率）；精轨=15-25秒调制动画（体验）

6.3 底卡系统（精轨·茶饮甜品）

  • 定义：底卡 = 茶具 + 底底，是制作茶饮/甜品的基础框架
  • 数量：69张，分四级梯度（市井13/雅士15/贡品14/上古27）
  • 槽位规则：类别约束型 —— 主鲜果槽 / 辅本草槽 / 花卉点缀槽
  • 组合数：每张底卡的可用组合数从8种到27种不等
  • 类型分布：茶饮花饮~32张、奶茶乳饮~10张、冷饮冰饮~5张、甜品冻/糕/布丁/仙饮共~22张

6.4 种植系统强化设计

# 强化方向 具体做法 价值
1 食材图鉴页丰富化 每种食材独立页面：古雅名+俗名 / 产地故事 / 古籍出处 / 药用或食用价值 / 关联诗词 / 助农信息 让每棵植物都有"人格"和故事
2 植株展示期 成熟期植株在小院内展示，可旋转查看；收获前是"观赏期" 种植=装饰，双重价值
3 种植随机变异 同种食材每次种植可能有微小差异（大小/颜色/特殊标签如"今年的桂花特别香"） 增加重复种植乐趣
4 助农实物联动 特定助农食材（赣南脐橙/阿克苏苹果等）种植达标后触发"丰收奖励"→ 真实水果寄送到家 "种什么得什么"核心承载功能
5 园丁排行榜 独立于茶艺榜的种植榜单：谁收集最多 / 等级最高 / 小院最美 给纯种植玩家社交出口
6 季节种植节奏 不同季节解锁不同食材种苗（呼应二十四节气） 让种植与时间产生意义关联
7 种苗交易 玩家之间互相赠送/交换种苗（替代"偷花"的社交方式） 增强种植社交维度

6.5 灵韵系统改造方向

原系统问题：规则复杂、付费门槛过高（约1000元起步）、耦合度低

改造方向： - 大幅简化：图标匹配替代复杂属性计算 - 降低门槛：首次付费从1000元降至6元级别 - 保留深度：免费可玩，付费加速/便览
