# 宋朝奶茶店·底卡与食材系统设计文档

> 版本 v2.0 · 以《梦华录》为美学参照，穿越到宋朝开奶茶店

---

## 一、世界观与设计出发点

### 背景设定

玩家扮演一位穿越到宋朝的现代人，在东京汴梁开一间茶饮铺子。宋朝是中国饮茶文化的顶峰——点茶、斗茶、茶百戏皆兴于此时——同时也是市井饮食最为丰富的朝代。玩家带着现代奶茶的配方记忆，在这个朝代"重新发明"饮品，从市集小摊一路做到贡品级别，乃至触碰上古传说的神秘配方。

### 美学参照：《梦华录》

《梦华录》剧中出现的饮品与甜点为本系统提供了直接参照：

| 类型 | 《梦华录》原型 | 在本游戏中的对应 |
|------|-------------|----------------|
| 点茶 | 径山茶、北苑先春斗茶 | 雅士级茶饮底卡 |
| 熟水 | 紫苏饮子（宋代第一名） | 草本类辅料核心食材 |
| 贡茶 | 龙凤团茶、青凤髓 | 贡品级茶饮底卡 |
| 冷饮 | 雪泡豆儿水、荔枝膏水 | 冰沙底、冰饮底卡 |
| 甜品果子 | 鲜花团子、桂花茶果、碧涧豆儿糕 | 甜品类底卡 |
| 穿越元素 | —（无原型） | 圆子奶茶底、黑糖乳底等 |

## 二、核心机制：底卡系统

### 什么是底卡？

底卡 = **茶具 + 底底**，是玩家制作饮品/甜品的基础框架。

类比：就像花艺游戏中"花瓶决定能插哪些花"，茶具+底底决定了这款饮品的风格，玩家通过在 3 个槽位中填入自己种植的食材来完成成品。

```
┌─────────────────────────────────────┐
│  汝窑天青盏 · 龙井底               │
│                                     │
│  槽位1 主鲜果  [枇杷][荔枝][雪梨]  │
│  槽位2 辅草本  [薄荷][藿香]        │
│  槽位3 花卉点缀[桂花][茉莉花]      │
│                                     │
│  可得组合数：3 × 2 × 2 = 12 种    │
└─────────────────────────────────────┘
```

### 设计原则

1. **底底排他**：定义底底风味的食材不会出现在槽位里。例如"荔浆底"的槽位中不会再出现荔枝，避免重复。
2. **每种食材只属于一张底卡**：稀缺性与唯一性保证了食材收集的动力。
3. **底卡解锁的是组合**：玩家解锁"汝窑天青盏·龙井底"这整个框架，而非分别解锁茶具和底底。

## 三、底卡梯度（共 69 张）

### 四个梯度

| 梯度 | 数量 | 槽位结构 | 可得组合数 | 定位 |
|------|:----:|---------|:---------:|------|
| 市井级 | **14** 张 | 主×3 + 辅×3 + 点×3 | 27 种 | 市集街头，日常滋养 |
| 雅士级 | 15 张 | 主×3 + 辅×3 + 点×2 | 18 种 | 文人雅集，品茗论道 |
| 贡品级 | **15** 张 | 主×3 + 辅×2 + 点×2 | 12 种 | 官宦贵族，贡品专享 |
| 上古传说级 | 27 张 | 主×2 + 辅×2 + 点×2 | 8 种 | 神话仙家·穿越秘方 |
| **合计** | **71 张** | — | — | — |

> 越高梯度的底卡，槽位选择越少，但食材越珍稀，视觉效果越华丽。

### 底卡类型分布

| 类型 | 数量 | 代表底卡 |
|------|:----:|---------|
| 茶饮/花饮/果饮 | ~32 张 | 定窑白瓷盏·酸梅汤底、汝窑天青盏·龙井底 |
| 奶茶/乳饮 | ~10 张 | 影青鼓腹杯·黑糖乳底、汝窑点茶盏·碾茶乳底 |
| 冷饮/冰饮 | ~5 张 | 竹编漆盖盅·冰镇消暑底、青瓷冰碗·冰沙底 |
| 甜品·冻 | ~10 张 | 磁州彩碟·藕粉冻底、官窑冰花碟·仙草冻底 |
| 甜品·糕 | ~5 张 | 竹笼蒸格·桂花糕底、磁州彩盘·茶香糕底 |
| **甜品·蜜饯** | **2 张** | **青釉小陶钵·糖渍蜜果底(T070)、鎏金镂空食盒·蜜饯雕花底(T071)** |
| 甜品·布丁/冰品 | ~4 张 | 水晶琉璃碗·星光布丁底、白玉莲碟·酥山极品底 |
| 仙饮/神话 | ~3 张 | 月宫玉碗·广寒清露底、昆仑玉爵·蟠桃仙底 |

### 穿越叙事线

27 张上古传说级底卡中，特设**穿越者秘方系列**（7 张），形成一条隐藏叙事：

```
市井级：竹编漆盖盅·冰镇消暑底
          ↓ 穿越者把冰镇技法带到市集
雅士级：磁州彩绘杯·草莓乳底 / 青瓷冰碗·冰沙底
          ↓ 新奇饮料被文人追捧
贡品级：官窑奶盖盏·奶沫顶底 / 汝窑点茶盏·碾茶乳底
          ↓ 惊动贵族，成为宫廷新宠
上古传说级：鎏金提把杯·圆子奶茶底（珍珠奶茶）
             漆雕食盒·杨枝甘露底
             玄铁保温爵·千年秘方底
```

## 四、食材系统（共 514 种）

### 三大类别

玩家在农场种植的作物只分三类，保持视觉清晰：

| 类别 | 数量 | 在底卡中的槽位角色 | 农场视觉特征 |
|------|:----:|------------------|------------|
| **鲜果类** | 188 种 | 主料（主鲜果） | 果树/灌木/藤蔓，按结果形态区分 |
| **花卉类** | 163 种 | 点缀 / 部分辅料 | 花型、颜色、花期各异 |
| **草本类** | 163 种 | 辅料（辅草本） | 叶型、株高、竹/藤/草形态区分 |

> **不包含**：谷物、豆类、根茎蔬菜。底卡的"红豆冻底""糯米糕底"只描述成品形态，不要求玩家种植豆粮。

### 四级稀有度

| 稀有度 | 食材特征 | 视觉表现 |
|--------|---------|---------|
| **市井级** | 常见品种，日常色泽 | 普通大小，饱满色彩 |
| **雅士级** | 特定产区/季节品种 | 异色变种（紫桃、白草莓）、独特形态 |
| **贡品级** | 宫廷贡品、名产精品 | 超大体型、异常光泽、珍贵品种名 |
| **上古传说级** | 神话仙家、穿越秘方 | 发光/结晶/仙气特效，名称带神话色彩 |

### 稀有度命名规范

**刻意避免**"御用X""贡品X"这类纯前缀命名——外观无法区分，收集感差。

| ❌ 不推荐 | ✅ 推荐方式 |
|----------|-----------|
| 御用梨 | 月华梨（仙气发光）/ 玉梨（翡翠色泽）/ 珍珠梨（形态特殊） |
| 贡品草莓 | 贡品草莓（宫廷产区可接受）/ 四季草莓（区分季节特性） |
| 御用薄荷 | 野生薄荷王（山野变种）/ 仙家薄荷露（传说效果） |

## 五、食材与底卡的匹配逻辑

### 槽位类型与食材类别对应

| 槽位类型 | 对应食材类别 | 出现底卡 |
|---------|------------|---------|
| 主鲜果 | 鲜果类 | 市井/雅士/贡品级大部分 |
| 主花卉 | 花卉类 | 花饮类底卡（晨露底、月季露底等）|
| 辅草本 | 草本类 | 大部分底卡的第2槽 |
| 辅花卉 | 花卉类 | 甜品底卡的第2槽（奶茶+花冻搭配）|
| 花卉点缀 | 花卉类 | 茶饮/果饮底卡的第3槽 |
| 草本点缀 | 草本类 | 甜品底卡的第3槽 |
| 珍稀鲜果/花卉/草本 | 对应类别贡品+ | 贡品级第2/3槽 |
| 传说XXX点缀 | 对应类别上古传说级 | 上古传说级第3槽 |

### 分配示例

**T004 越窑青瓷瓯·薄荷露底（市井级）**
```
排除规则：薄荷相关食材不出现在槽位中
槽位1 主鲜果×3：蟠桃 / 青李 / 红李
槽位2 辅草本×3：鼠尾草 / 马鞭草 / 香茅
槽位3 花卉点缀×3：梅花 / 桃花 / 杏花
可得组合：27 种
```

**T041 白玉浅碟·桂花琼冻底（贡品级·甜品）**
```
排除规则：桂花/金桂/银桂/丹桂不出现在槽位中
槽位1 主鲜果×3：贡品级水果（如玛瑙樱桃、御苑樱桃、贡品雪梨）
槽位2 辅花卉×2：贡品级花卉（如碧玉荷花、红莲）
槽位3 珍稀草本点缀×2：贡品级草本（如贡品百合草、御用麦冬草）
可得组合：12 种
```

**T051 鎏金提把杯·圆子奶茶底（上古传说级·穿越）**
```
排除规则：无
槽位1 珍稀鲜果×2：传说级浆果（如不老莓、天外来果）
槽位2 珍稀草本×2：传说级香草（如九转还魂草、玄冥药草）
槽位3 传说花卉点缀×2：传说级花卉（如天山雪莲花、神农灵芝花）
可得组合：8 种
风味描述：糯米圆子沉底，以竹管直饮，食客目瞪口呆，称其"未来之饮"
```

## 七、农场视觉设计原则

为保证同类食材在农场中可区分，每类采用不同植株形态：

### 鲜果类（8种植株形态）

| 形态 | 代表植物 | 视觉特征 |
|------|---------|---------|
| 大乔木 | 荔枝、龙眼、柿子 | 圆冠大树，果实挂枝 |
| 小乔木 | 桃、梅、杏、枇杷 | 开花结果小树 |
| 灌木 | 山楂、石榴、林檎 | 矮株，果实明显 |
| 攀藤 | 葡萄、覆盆子 | 架上蔓延，串状果 |
| 地生浆果 | 草莓、野草莓 | 贴地低矮，色泽鲜艳 |
| 坚果 | 板栗（刺球苞）、核桃（纹壳）、银杏（扇叶） | 各有特殊形态 |
| 热带型 | 甘蔗（高挺分节）、佛手柑（手指状） | 轮廓极特殊 |
| 水生型 | 莲子（荷叶+莲蓬） | 水面叶片+立挺花茎 |

### 花卉类（6种植株形态）

地被花草 → 直立草本花 → 灌木花 → 攀援花藤 → 乔木花 → 水生花

### 草本类（7种植株形态）

匍匐草本（薄荷）→ 直立草本（紫苏）→ 竹型（各色竹）→ 高挺禾本（柠檬草）→ 莲座型（芦荟）→ 蔗茎型（甘蔗叶）→ 香料灌木（肉桂）
